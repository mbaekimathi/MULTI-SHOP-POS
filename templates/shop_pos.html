Set up printer

Applies only to {{ shop.shop_name }}. Other shops use their own saved printer.

Checkout printing: each sale opens the browser print dialog (same as Ctrl+P) with a standard thermal-width receipt (80mm or 58mm per IT receipt settings) and full line items. Connection type

Windows “paired” is not the same as POS linked. This screen uses Chrome/Edge Bluetooth Low Energy. Keep the printer powered on and in range, then use Connect printer. If it never turns green, use USB or print from the browser dialog instead.

Checking…
Printer not showing in the list?

This flow needs a printer that speaks Bluetooth Low Energy (BLE). Older models that only use “Classic” Bluetooth Pairing typically do not appear in Chrome — switch to Network or USB above instead.

Thermal · ESC/POS is sent to the printer’s raw port (often 9100) via this server. The machine running the app must be on the same LAN as the printer. Opening this screen runs an automatic scan (live progress; includes common subnets like 192.168.100.x); or scan a single subnet or type a printer IP or hostname to check.

LAN scans try this port first, then other common thermal/raw ports (9101, 4000, 5000, …).

LAN print agent (cloud hosting)

If this site is hosted off-site, run the Python agent on a computer on the same Wi‑Fi as the thermal printer. Receipts queue in the database; the agent pulls them and prints locally.

USB printing is turned off in company/shop settings. Ask IT to enable USB under printing options, or use Network / Bluetooth.

Use Chrome or Edge. Plug in USB, tap Choose USB printer, and pick your device. On Windows with a POS-80C driver, checkout prints through the browser dialog (same as Ctrl+P). See “Printer not in the list?” for details.

Checking USB…

Printer not in the list?
  • “Access denied” on Windows: Normal when POS-80C is installed — save the USB device anyway; receipts print via browser (Destination: POS-80C).
  • Install the Windows driver for your thermal printer, then reconnect USB.
  • Close other apps that may lock the printer (old POS, label tools).
  • If only the browser dialog works, set POS-80C as the default printer in Windows and use 80 mm paper in More settings.
