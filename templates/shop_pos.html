Receipt printer

Set up printer

Applies only to {{ shop.shop_name }}. Other shops use their own saved printer.

Checkout printing: each sale opens the browser print dialog (same as Ctrl+P) with a standard thermal-width receipt (80mm or 58mm per IT receipt settings) and full line items. Under Destination, pick your POS-80 / thermal printer (not Save as PDF). Optional settings below are for advanced pairing only.

Connection type

For receipt roll / thermal printers: ESC/POS is sent over Bluetooth (not the system print dialog). Pair once — it saves. If it stops working, use Forget saved printer and pair again.

Thermal · ESC/POS is sent to the printer’s raw port (often 9100) via this server. The machine running the app must be on the same LAN as the printer. Opening this screen runs an automatic scan (live progress; includes common subnets like 192.168.100.x); or scan a single subnet or type a printer IP or hostname to check.

LAN scans try this port first, then other common thermal/raw ports (9101, 4000, 5000, …).
